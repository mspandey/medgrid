# Screenshots Folder

- `landing.png` — Landing page
- `dashboard.png` — Hospital dashboard/search
- `staff_portal.png` — Staff portal (beds/blood inventory)
- `blood_donation.png` — Blood donation form
- `chatbot.png` — AI chatbot
